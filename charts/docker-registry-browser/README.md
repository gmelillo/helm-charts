**Homepage:** <https://github.com/gmelillo/helm-charts>

## Installation

### Add Helm repository

```shell
helm repo add gmelillo https://helm.melillo.me
helm repo update
```

### Install

```shell
helm install --generate-name gmelillo/docker-registry-browser
```
